# TripNest Travels ✈️🌍

TripNest Travels is a travel booking website developed to provide users with an easy and interactive travel planning experience. The project includes destination booking, booking confirmation, ticket viewing, and booking history features with a responsive user interface.

## Features

* Travel destination booking form
* Booking confirmation page
* My Bookings / booking history section
* Ticket viewer page
* Dummy payment interface for booking flow
* Dark mode support
* Responsive design for different screen sizes

## Technologies Used

* HTML
* CSS
* JavaScript
* Tailwind CSS
* Node.js
* Express.js

## Project Files

* `index.html` → Main homepage
* `booking-form.html` → Travel booking form
* `booking-confirmation.html` → Booking confirmation page
* `my-bookings.html` → User booking history
* `ticket-viewer.html` → Ticket display page
* `script.js` → Main frontend logic
* `booking-form-script.js` → Booking form handling
* `darktheme.js` → Dark theme functionality
* `server.js` → Backend server
* `models/User.js` → User data model
* `models/Booking.js` → Booking data model

## How to Run

1. Open project in VS Code
2. Install dependencies:

```bash
npm install
```

3. Start server:

```bash
node server.js
```

4. Open browser and run locally
